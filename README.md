# HTML div

split screen into div

## Installation

```bash
Clone the repo: get clone https://github.com/dahoonchoi/html_div.git
```

## Usage

```bash
Install 'html' 
```

## Contributing
```
Html and 'css' were split to initialize and 'flex' and 'overflow' were used in 'div' to enable dynamic conversion.

*{
    padding: 0;
    margin: 0;
}
html,body{
    text-align: center; 
    width: 100%;
    height: 100%;
    min-height: 100%;
    overflow-x: hidden;
}
.division{
    overflow: auto;
    display: flex;
    flex-direction: row;
    min-height: 83%;
}
```
